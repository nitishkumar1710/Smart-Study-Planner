# Smart Study Planner

## Overview
An AI-powered smart study planner that helps users manage schedules intelligently using ANN models and LLM-based reasoning.

## Features
- Intelligent task scheduling
- Habit learning using ANN
- Real-time updates using WebSockets
- Personalized study planning

## Tech Stack
- React.js
- Node.js
- MongoDB
- WebSockets
- Python
- Machine Learning

## Future Improvements
- Voice assistant integration
- Productivity analytics
- Mobile application support
